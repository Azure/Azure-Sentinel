Microsoft Sentinel Solutions Analyzer — Now Interactive! 🔍

The Solutions Analyzer documentation just got a major upgrade — a fully interactive index page with instant filtering, sorting, and search across every entity type.

Six browsable views let you explore the full Sentinel ecosystem:
📦 Solutions · 🔌 Connectors · 📋 Tables · 📄 Content · 🔧 Parsers · 🔄 ASIM Parsers

What makes it useful:

🔎 Per-column filters — Every column has its own filter. Combine them to answer questions like "Which active solutions use OAuth2 connectors?" or "Which deprecated connectors still have associated content?"

👆 Click-to-filter — Click any cell value to instantly filter by it. See a connector method you're interested in? One click narrows the view.

🔗 Deep links — Every entity name links to its full documentation page with properties, related items, setup instructions, and more.

↕️ Column sorting — Click any column header to sort ascending or descending. Quickly find the newest solutions, busiest tables, or largest content collections.

🔍 Global search — Find anything across all columns instantly.
